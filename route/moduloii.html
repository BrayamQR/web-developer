UNIDAD DE COMPETENCIA N° 02

Desarrollar la construcción de programas de los sistemas de información, de acuerdo al diseño funcional, estándares internacionales de TI, buenas prácticas de programación y políticas de seguridad de la organización.

UNIDADES DIDÁCTICAS ESPECÍFICAS

Curso Descripción Creditos Horas
Taller de Programación Web Diseñar programas según los requerimientos del cliente, utilizando las buenas prácticas del desarrollo de aplicaciones. 4 7
Base de Datos Implementar la base de datos de acuerdo a la arquitectura de los sistemas de información. 3 4
Lenguaje de programación Desarrollar un aplicativo web basado en el entorno del cliente, teniendo en cuenta el diseño de sistemas, buenas practicas y cilco de vida de desarrollo de sistemas. 5 8
Requerimiento de Software Analizar y diseñar el sistemas y determinar los requerimientos necesarios del sistema. 3 4
Pruebas de Software Implementar plan de pruebas unitarias de los componentes informáticos de acuerdo a los requerimientos del diseño. 3 4
Diseño de Software Identificar y analizar metodologías de desarrollo de software para documentar los sistemas de información. 2 3
Integración y Testeo Planificar y ejecutar los planes de integración y testeo. 2 3
Desarrollo Web Desarrollar un aplicativo web, teniendo en cuenta la arquitectura del software, buenas prácticas y ciclo de vida de desarrollo de sistemas. 5 8
Arquitectura de Soluciones Web Documentar un proyecto web basado en el entorno del cliente servidor teniendo en cuenta el diseño de sistemas, buenas prácticas y ciclo de vida de desarrollo de sistemas. 5 8
Administración de Base de Datos Implementar la base de datos de acuerdo a la arquitectura de los sistemas de información. 4 5

Capacidades del estudiante
  • Diseña base de datos de una organización.
  • Elabora los requerimientos y el diseño del modelo de un sistema de información.
  • Implementa un sistema de información en diferentes arquitecturas utilizando un lenguaje de programación.
  • Realizan pruebas, analiza e interpreta los resultados para implementar mejoras en aplicaciones y base de datos.
